Using Modern Webapp

Getting Started

Installation

The app can be installed by cloning the main repository. 

git clone git@github.com:danielstern/modern-webapp.git;

Running the App

The app is run on node using grunt. Run npm install followed by bower install and grunt. 

npm install;
bower install;
grunt serve;

Grunt Modes

Use grunt serve for development. grunt serve:dist can be used to test the build. grunt:build will create a fully built version in the /dist folder that can be used elsewhere. 

grunt serve; // runs the app locally
grunt serve:dist; // runs the app locally in its built form
grunt build; // builds the app and puts it in the /dist folder

Deployment

The app can either be deployed by running build, then exporting the contents of the /dist directory to the endpoint of your choice, or by deploying to heroku.

Heroku Deployment

heroku create;
git push heroku master;
