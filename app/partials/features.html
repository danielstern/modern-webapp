Features

LESS

LESS is applied automatically to styles in the /less folder. CSS styles can also be placed in this folder, but they must have a .less extension. Build is run automatically while grunt serve is running.

LESS source maps are available automatically.

TypeScript

TypeScript files are located in the /typescript directory and are compiled to javascript automatically. JavaScript files can also go in this directory.

TypeScript source maps are available automatically.

Ui-Router

Routing is handled in real time with ui-router. Routes can be added in typescript/main.ts
